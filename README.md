## What It Does
Have you ever thought about how we would communicate with other species? How would we interact with them face to face? This project is a Maritian-Human translator webpage/app, it is designed to provide users help communicate with other species, the translator converts English text into a readable Martian format that can be instantly interpreted, same applies for the Martian language to English. The app provides many features that allows the Martian and Humans to interact as if they spoke the same language, these features/tools create a more simple, clean, and friendly way between interactions with Humans and Martians. Additionally, the webpage/app will allow users to subscribe the translator to unlock all the features and tools, with the first month being free users get access to the glimpse of what it is like to talk to Martians on a day to day basis.
